# Sistema de Turnos para Enfermería Escolar

Este proyecto consiste en un sistema básico de gestión de turnos desarrollado en Python para organizar la atención de estudiantes en una enfermería escolar desde la consola.
## El programa permite:
1. Registrar estudiantes en una fila de atención.
2. Asignar turnos automáticamente.
3. Llamar al siguiente estudiante.
4. Visualizar los turnos pendientes.

El objetivo principal es mejorar el orden y la eficiencia en el proceso de atención, evitando confusiones y largas filas.

## Tecnologías utilizadas
1. Python 3
2. Visual Studio Code

## Funcionalidades
1. Sacar turno : El usuario ingresa el nombre del estudiante y el sistema asigna automáticamente un número de turno.

2. Llamar siguiente turno : El sistema atiende al primer estudiante registrado, siguiendo el orden de llegada.

3. Ver turnos pendientes : Permite visualizar todos los estudiantes que aún no han sido atendidos.

4. Salir : Finaliza la ejecución del programa.

## Cómo ejecutar
1. Descargar o copiar el archivo .py
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar: python turnos.py o python3 turnos.py
